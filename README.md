# Online Book Store Schema

## Objective
Design a database schema for an Online Book Store using ER Diagrams, Relationships, and Normalization.

## Tools Used
- MySQL
- MySQL Workbench
- Draw.io / Lucidchart

## Entities
- Customers
- Books
- Orders
- OrderItems

## Relationships
- One Customer can place many Orders.
- One Order can contain many Books.
- One Book can appear in many Orders.

## Features
- Customer Management
- Book Inventory Management
- Order Processing
- Order Tracking
